### 核心理念

**概念关系连接**：每个文件用 → 显示概念间的关系

```
用户 → 终端 → Shell → 内核 → 硬件
代码 → 编译器/解释器 → 运行时 → 操作系统
前端 → API → 后端 → 数据库
```
