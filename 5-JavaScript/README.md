# JavaScript

> 全栈开发的必备语言 — 从浏览器到服务器，一门语言搞定一切

## 核心特点

- **跨平台**：浏览器、服务器（Node.js）、移动端、桌面端全覆盖
- **事件驱动**：非阻塞异步编程，支持实时应用
- **弱类型**：灵活简洁，上手容易
- **生态丰富**：npm 拥有全球最大的开源库生态
- **入门友好**：浏览器即可运行，无需复杂配置

## 适用场景

`前端开发` `后端服务` `移动应用` `桌面软件` `工具脚本` `自动化`

## 学习路线

```
简介 → 基础语法 → 常用操作 → 高级用法 → 框架实战
```
