# JavaScript 开发环境

## 浏览器开发者工具

学习 JavaScript 最简单的方式——无需安装任何软件，直接用浏览器！

1. 在任意网页上按 **F12** 打开开发者工具
2. 切换到 **Console（控制台）** 标签
3. 直接输入 JavaScript 代码并按 Enter 执行

```
常用快捷键：
- Chrome/Edge：F12 或 Ctrl+Shift+I
- Firefox：Ctrl+Shift+K
- Mac：Cmd+Option+I
```

## Node.js 环境

Node.js 是服务器端 JavaScript 运行环境，也常用于本地开发。

**安装步骤**：
1. 访问 https://nodejs.org 下载 LTS（长期支持版）
2. 安装后，打开终端/命令提示符
3. 输入 `node -v` 验证安装成功

**运行 JavaScript 文件**：
```bash
node hello.js
```

**交互式 REPL**：
```bash
node
> console.log('Hello')
Hello
```

## 浏览器控制台 vs Node.js

| 环境 | 启动方式 | 可用 API |
|------|---------|---------|
| 浏览器控制台 | F12 → Console | DOM、BOM、fetch |
| Node.js | 终端输入 node | 文件系统、网络、操作系统 |

## 轻量级编辑器

如果不需要 VS Code 的完整功能，可以先用轻量级工具入门：

- **浏览器控制台**：零配置，立即体验
- **CodePen / JSFiddle**：在线代码编辑器，无需安装
- **Node.js REPL**：学习服务器端语法
