# JavaScript 应用场景

## 前端开发

JavaScript 最核心的应用领域。通过操作 DOM（文档对象模型）和处理用户事件，实现网页的动态交互效果。

```javascript
// DOM 操作示例
document.querySelector('h1').textContent = 'Hello, JavaScript!';

// 事件处理示例
button.addEventListener('click', function() {
    alert('按钮被点击了！');
});
```

主流框架：React、Vue.js、Angular

## 后端开发

Node.js 让 JavaScript 可以在服务器端运行，适合构建高性能网络服务。

```javascript
// Node.js 创建 HTTP 服务器
const http = require('http');
http.createServer((req, res) => {
    res.writeHead(200);
    res.end('Hello from Node.js!');
}).listen(3000);
```

典型场景：RESTful API、实时通信（WebSocket）、微服务

## 移动端开发

跨平台移动开发框架让 JavaScript 可以开发 iOS 和 Android 原生应用。

- **React Native**：Facebook 推出的跨平台框架
- **uni-app**：基于 Vue.js 的跨平台框架
- **Flutter**（Dart 语言，但理念类似）

## 桌面端应用

Electron 框架允许使用 Web 技术（HTML + CSS + JavaScript）开发跨平台桌面应用。

著名案例：VS Code、Slack、GitHub Desktop、Discord

## 工具脚本与自动化

JavaScript 可以编写各种脚本，实现自动化操作：

- 前端构建工具（Webpack、Vite、Rollup）
- 代码检查工具（ESLint）
- 自动化测试框架（Jest、Mocha）
- 脚本任务（npm scripts）

## 全栈开发

JavaScript 是少有的能贯通前后端的语言。一个全栈开发者可以用同一门语言完成：

```
前端（React/Vue） → 后端（Node.js/Express） → 数据库（MongoDB）
```

这大大降低了学习成本，是 JavaScript 独特的优势。
