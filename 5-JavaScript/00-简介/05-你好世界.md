# 你好，世界！

让我们编写第一个 JavaScript 程序，体验这门语言的简洁与强大。

## 浏览器环境

### 方法一：使用开发者工具控制台

1. 打开浏览器，访问任意网页
2. 按 F12 打开开发者工具
3. 点击 Console 标签
4. 输入以下代码，按 Enter：

```javascript
console.log('你好，世界！');
```

你会看到控制台立即输出：`你好，世界！`

### 方法二：网页内嵌脚本

```html
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <h1>你好，世界！</h1>
    <script>
        console.log('JavaScript 开始执行');
        alert('欢迎来到 JavaScript 世界！');
        console.log('执行完毕');
    </script>
</body>
</html>
```

## Node.js 环境

创建一个名为 `hello.js` 的文件：

```javascript
// 这是一个注释，以 // 开头
/* 这也是注释，
   可以写多行 */

console.log('你好，世界！');
```

打开终端，运行：

```bash
node hello.js
```

你会看到终端输出：`你好，世界！`

## 代码逐行解释

```javascript
console.log('你好，世界！');
//   ^^^^^ 方法名（函数）
//        ^^^^ 参数：要打印的内容
//            ^^^^^^^^^^^ 分号，标识语句结束（ES6 后可省略）
```

| 代码部分 | 含义 |
|---------|------|
| `console` | 控制台对象 |
| `.log()` | 在控制台打印信息的方法 |
| `'你好，世界！'` | 字符串，用引号包裹的文本 |
| `;` | 语句结束符 |

## 常见错误

```javascript
// 错误：使用了中文引号或缺少引号
console.log('你好');   // 错误：使用了中文单引号 ''
console.log(你好);     // 错误：缺少引号，JavaScript 会当成变量名

// 正确：使用英文引号
console.log("你好");   // 双引号
console.log('你好');   // 单引号
console.log(`你好`);   // 反引号（模板字符串）
```
