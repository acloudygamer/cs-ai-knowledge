# JavaScript 语言简介

## 历史背景

1995年，网景公司（Netscape）为了在浏览器中实现动态交互功能，委托 Brendan Eich 在短短10天内设计了一门脚本语言。最初命名为 Mocha，随后更名为 LiveScript，最终因为与 Java 的合作营销协议而改名为 JavaScript。

## JavaScript 与 ECMAScript

很多人分不清 JavaScript 和 ECMAScript 的关系：

- **JavaScript** 是由 Mozilla 主导开发的编程语言实现
- **ECMAScript** 是由 ECMA International 制定的语言标准（ES5、ES6、ES2020等）
- 可以理解为：ECMAScript 是"语法规范"，JavaScript 是"符合规范的实现"

## 浏览器中的 JS vs Node.js 中的 JS

| 特性 | 浏览器 JavaScript | Node.js JavaScript |
|------|------------------|-------------------|
| 执行环境 | 浏览器 | 服务器（命令行） |
| API 差异 | DOM 操作、BOM API | 文件系统、网络、操作系统 |
| 模块系统 | ES Modules、CommonJS | ES Modules、CommonJS |
| 用途 | 网页交互、表单验证 | 后端服务、工具脚本、桌面应用 |

两者都遵循 ECMAScript 标准，核心语法完全一致。

## 版本演进

- **1997年**：ES1 — 首个标准化版本
- **2009年**：ES5 — 引入 JSON、严格模式
- **2015年**：ES6（ES2015）— 重大更新，引入 class、Promise、箭头函数
- **2015年后**：每年一个新版本（ES2016、ES2017...）
