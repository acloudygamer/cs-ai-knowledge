# JavaScript 学习路线

## 版本演进概览

现代 JavaScript 以 **ES6（ECMAScript 2015）** 为分水岭：

| 阶段 | 版本 | 特点 |
|------|------|------|
| 传统 JS | ES5 及以前 | 回调地狱、作用域混乱 |
| 现代 JS | ES6+ | class、Promise、箭头函数、模块化 |
| 最新 JS | ES2020+ | 可选链、空值合并、BigInt |

**建议**：直接学习 ES6+ 版本，避免再去研究 var、prototype 等旧语法。

## 学习阶段规划

### 第一阶段：语言基础（1-2周）

```
变量与数据类型 → 运算符 → 条件语句 → 循环 → 函数 → 数组 → 对象
```

### 第二阶段：异步编程（1周）

```
回调函数 → Promise → async/await → 事件循环
```

### 第三阶段：DOM 与浏览器 API（1-2周）

```
DOM 操作 → 事件处理 → 动画 → 表单验证 → fetch API
```

## 前端框架选择

| 框架 | 特点 | 学习难度 | 适用场景 |
|------|------|---------|---------|
| React | 组件化、Hooks、生态强大 | 中等 | 中大型应用 |
| Vue | 简洁易学、文档友好 | 较低 | 中小型应用 |
| Angular | 完整解决方案、TypeScript优先 | 较高 | 企业级应用 |

**建议**：
- 新手首选 **Vue**（上手快）
- 想进大厂/生态丰富 → **React**
- 喜欢强类型/企业级 → **Angular**

## Node.js 后端路线

```
Node.js 基础 → Express/Koa 框架 → 路由与中间件 →
数据库（MongoDB/MySQL）→ RESTful API → 认证与授权
```
