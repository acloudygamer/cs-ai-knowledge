# JavaScript 语言特点

## 事件驱动编程

JavaScript 在浏览器中以事件驱动方式运行。代码不按顺序逐行执行到底，而是"等"着，当用户点击按钮、网络请求完成、页面加载完毕等事件发生时，相应的处理函数才会被调用。

```javascript
// 点击按钮时才会执行这个函数
document.getElementById('btn').addEventListener('click', function() {
    console.log('按钮被点击了！');
});
```

## 单线程异步模型

JavaScript 是单线程语言，但通过事件循环（Event Loop）实现非阻塞异步操作。当遇到耗时操作（如读取文件、发送网络请求）时，不会傻傻地等待，而是注册回调函数后继续执行其他代码。

```javascript
console.log('开始');
setTimeout(function() {
    console.log('异步任务完成了');
}, 1000);
console.log('结束');
// 输出顺序：开始 → 结束 → 异步任务完成了
```

## 弱类型

JavaScript 是弱类型语言，允许隐式类型转换（不同类型运算时自动转换）。

```javascript
console.log('10' + 5);  // '105'（数字被转成字符串）
console.log('10' - 5);   // 5（字符串被转成数字做减法）
console.log(true + 1);   // 2（true 转成 1）
```

## 动态类型

JavaScript 是动态类型语言，变量类型在运行时确定，可以随时改变。

```javascript
let x = 10;        // number
x = 'hello';       // 自动变成 string
x = true;           // 又变成 boolean
```

## 跨平台

JavaScript 几乎可以在任何设备上运行：

- 浏览器（Chrome、Firefox、Safari、Edge）
- 服务器（Node.js）
- 移动端（React Native、uni-app）
- 桌面端（Electron）
- 嵌入式设备

## 原型继承

JavaScript 采用原型继承而非类继承（虽然 ES6 引入了 class 语法糖）。每个对象都有一个原型对象，通过原型链实现属性和方法的共享。

```javascript
// 原型链：car → Vehicle.prototype → Object.prototype → null
const car = { wheels: 4 };
const sportsCar = Object.create(car);
sportsCar.speed = 'fast';

console.log(sportsCar.wheels);    // 4（从原型继承）
console.log(sportsCar.hasOwnProperty('wheels')); // false
```
