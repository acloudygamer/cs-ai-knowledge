# AI时代开发指南

## AI辅助编程
- **工具选择**：Claude Code / GitHub Copilot / Cursor
- **使用场景**：代码补全、bug修复、知识问答、重构
- **最佳实践**：给AI清晰的上下文、分步骤解决复杂问题
- **代码示例**：如何让AI帮你写单元测试

## 提示词工程
- **结构化prompt**：Role + Task + Context + Output Format
- **迭代优化**：根据输出调整prompt
- **代码示例**：不同场景的prompt模板

## AI代码审查
- **审查流程**：AI预审 → 人工复核 → 修改
- **适用场景**：代码风格、潜在bug、安全问题
- **工具**：Claude review、CodeRabbit、GPT review
- **代码示例**：如何让AI审查你的代码

## 实践建议
- 每天用AI完成3个小任务练手
- 重要代码不要完全依赖AI，自己要理解
