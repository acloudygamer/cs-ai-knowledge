# Java

**企业级开发的主流选择**——全球超过 3000 万开发者，Spring 生态统治后端，Android 开发基石，大数据领域核心语言。

### 核心特点

- **平台无关**：JVM 字节码机制，真正 "一次编写，到处运行"
- **自动内存管理**：GC 垃圾回收，告别手动内存管理
- **面向对象**：纯 OOP 语言，代码结构清晰易维护
- **生态成熟**：Spring 全家桶、Maven/Gradle、海量类库
- **多线程支持**：内置多线程，虚拟线程让并发更简单
- **稳定可靠**：银行、电商、政府系统首选语言

### 适用场景

`企业级后端` `Android 开发` `Web 服务` `微服务` `大数据` `金融系统`

### 学习路线

Java 8（Lambda/Stream）→ Java 11（HttpClient）→ Java 17（密封类）→ Java 21（虚拟线程）
