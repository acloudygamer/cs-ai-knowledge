# Java 开发环境

> 本章节提供快速上手指南。如需详细的安装配置说明，请参考：[安装与环境](../01-基础/01-安装与环境.md)

## 快速安装 JDK

### 推荐方式：Eclipse Temurin（原 AdoptOpenJDK）

1. 访问 https://adoptium.net/
2. 选择最新 LTS 版本（目前为 Java 21 或 Java 17）
3. 下载对应平台的安装包（Windows/macOS/Linux）
4. 运行安装程序，默认安装即可

### 验证安装

打开终端（Windows 用 PowerShell 或 CMD），输入：

```bash
java -version
javac -version
```

看到类似以下输出即表示安装成功：

```
openjdk version "21.0.2" 2024-01-16
```

## IDE 选择

### IntelliJ IDEA（推荐）

- **下载**：https://www.jetbrains.com/idea/download/
- **版本**：Community（社区版，免费）或 Ultimate（付费，功能更全）
- **特点**：智能代码补全、重构工具、集成 Maven/Gradle、对 Java 支持最完善

### VS Code

- 安装 **Extension Pack for Java** 插件
- 适合轻量级项目和对 VS Code 熟悉的开发者
- **下载**：https://code.visualstudio.com/

### 不推荐 Eclipse

Eclipse 虽然经典，但相比 IDEA 体验差距较大，新手建议直接使用 IDEA 社区版。

## 创建第一个项目（IntelliJ IDEA）

1. 打开 IDEA → **New Project**
2. 选择 **Java** → 选择 JDK 版本 → 点击 **Next**
3. 输入项目名称和保存位置 → **Create**
4. 右键 `src` 文件夹 → **New** → **Java Class**
5. 输入类名（如 `HelloWorld`），IDEA 会自动生成 `main` 方法模板

## 构建工具（补充知识）

当你开始中等规模项目时，会需要构建工具：

- **Maven**：配置文件 `pom.xml`，依赖管理成熟稳定
- **Gradle**：配置文件 `build.gradle`，语法更简洁

初学者可以先不学这两个工具，在完成基础语法学习后再接触即可。
