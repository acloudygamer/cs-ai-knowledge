# Java 语言特点

## 1. 纯面向对象

Java 是 "纯正" 的面向对象语言，所有代码都必须在类中定义（即使是 `main` 方法）。它不支持独立的函数、全局变量，必须通过类和对象来组织代码。

这带来了：
- 代码结构清晰，容易模块化
- 便于复用和维护
- 设计模式在 Java 世界里应用广泛

## 2. 平台无关

通过 JVM 实现真正的跨平台：
- `.java` 源文件 → `javac` 编译 → `.class` 字节码
- `.class` 文件在任何安装了 JVM 的平台上都能运行
- 虚拟机负责平台相关的细节处理

## 3. 自动内存管理

Java 的 **GC（Garbage Collection，垃圾回收）** 机制会自动：
- 分配内存给新对象
- 识别不再使用的对象
- 释放这些对象的内存

开发者无需手动 `free` 或 `delete`，大大减少了内存泄漏和野指针问题。

## 4. 健壮性与安全性

- **强类型检查**：编译时严格检查类型错误
- **异常处理**：统一的异常机制代替传统的错误码
- **安全管理器**：可控制代码的访问权限（如禁止读写本地文件）

## 5. 多线程支持

Java 从诞生起就内置了多线程支持：
- `Thread` 类、`Runnable` 接口
- `synchronized` 关键字实现同步
- `java.util.concurrent` 包提供高级并发工具

现代 Java（17+）还引入了 **虚拟线程（Virtual Threads）**，让并发编程更加简单。

## 6. 丰富的生态系统

- **企业级框架**：Spring、Spring Boot、Hibernate
- **构建工具**：Maven、Gradle
- **海量类库**：Apache Commons、Google Guava 等
