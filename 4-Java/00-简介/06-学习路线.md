# Java 学习路线

## Java 版本演进

了解 Java 的发展历史有助于理解语言特性的出现背景：

| 版本 | 发布年份 | 重要特性 |
|------|----------|----------|
| Java 8 | 2014 | **Lambda 表达式**、Stream API、Optional、新日期时间 API |
| Java 11 | 2018 | **HttpClient API**（正式版）、局部变量类型推断 `var`、直接运行单文件源码 |
| Java 17 | 2021 | **密封类**（Sealed Classes）、Switch 表达式（正式版）、随机数生成器改进 |
| Java 21 | 2023 | **虚拟线程**（Virtual Threads）、Record 模式、String Templates |

> **建议初学者以 Java 17 或 21 为主版本**，因为它们是长期支持版（LTS），而且包含了现代语言特性。

## 推荐学习顺序

### 第一阶段：语法基础（约 2-4 周）

1. 变量、数据类型、运算符
2. 条件判断（if/else、switch）
3. 循环（for、while、do-while）
4. 数组
5. 方法（函数）定义与调用

### 第二阶段：面向对象（约 2-3 周）

1. 类与对象
2. 封装、继承、多态
3. 接口与抽象类
4. 异常处理（try/catch/finally）
5. 常用类（String、List、Map）

### 第三阶段：工具与实践（约 2-3 周）

1. **Maven 或 Gradle** 构建工具
2. **JUnit** 单元测试
3. **Git** 版本控制
4. 基础数据结构（集合框架）
5. 文件操作与 I/O

### 第四阶段：Web 开发（约 4-8 周）

1. **Spring Boot** 快速入门
2. RESTful API 设计
3. 数据库操作（JdbcTemplate 或 MyBatis）
4. Spring MVC 请求处理流程
5. 基础前端知识（HTML/CSS/JavaScript）

### 第五阶段：进阶与扩展

- Spring Cloud（微服务）
- 中间件（Redis、Kafka）
- 性能调优
- JVM 原理

## 推荐学习资源

### 书籍

- **《Java 核心技术·卷 I》**（Cay S. Horstmann）— 经典入门书籍
- **《Effective Java》**（Joshua Bloch）— Java 最佳实践，提升代码质量

### 在线学习

- **B 站**：搜索 "Java 入门" 有大量优质视频教程
- **Oracle 官方教程**：https://docs.oracle.com/javase/tutorial/
- **Spring 官方文档**：https://spring.io/projects/spring-boot

### 练习平台

- **LeetCode**（算法与数据结构）
- **洛谷**（算法竞赛，适合巩固基础）

## 学习建议

1. **多动手敲代码**：只看不动手等于没学
2. **从错误中学习**：遇到编译错误是正常的，耐心分析报错信息
3. **不要死记语法**：理解背后的原理更重要
4. **尽早接触项目**：学完面向对象后可以尝试做个小项目（如通讯录管理系统）
5. **养成阅读源码的习惯**：看优秀的开源项目是怎么写的
