# Go

**云原生时代的高效语言** — 简洁如 Python，性能如 C，并发如 goroutine。

## 核心特点

- **静态类型 + 类型推导** — 安全与简洁并存
- **goroutine 原生并发** — 轻量级协程，万级并发
- **编译飞快** — 毫秒级编译整个项目
- **自动垃圾回收** — 告别手动内存管理
- **单一可执行文件** — 部署零依赖
