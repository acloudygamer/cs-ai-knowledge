# Go 开发环境

## 安装 Go

推荐使用 [go install](https://go.dev/doc/install) 方式安装（适用于 Go 1.17+）：

```bash
# 安装最新稳定版
go install golang.org/dl/go1.22.0@latest

# 使用新版本
go1.22.0 download
go1.22.0 version
```

验证安装成功：

```bash
go version
# go version go1.22.0 linux/amd64
```

## 编辑器配置

### VS Code

安装扩展 **Go**（官方出品），然后按 `Ctrl+Shift+P`，输入 "Go: Install/Update Tools"，全选安装。

### GoLand

JetBrains 出品，对 Go 支持最全面。安装后无需额外配置，开箱即用。

## 项目初始化

```bash
# 创建模块（项目）
go mod init github.com/你的用户名/项目名

# 运行程序
go run main.go

# 编译为可执行文件
go build -o myapp main.go
```

## 常用命令速查

| 命令 | 作用 |
|------|------|
| go mod init | 初始化新模块 |
| go run | 运行程序 |
| go build | 编译程序 |
| go test | 运行测试 |
| go get | 安装依赖包 |
| go mod tidy | 清理未使用依赖 |

> 详细的 IDE 配置、 GOPATH 、环境变量说明，请参考 [01-基础/01-安装与环境.md](../01-基础/01-安装与环境.md)。
