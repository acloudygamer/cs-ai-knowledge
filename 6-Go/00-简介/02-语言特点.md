# Go 语言特点

## 静态类型，动态简洁

Go 是静态类型语言，但写起来像动态语言。变量声明简洁，类型推导让你无需处处标注类型，却仍能享受类型安全的好处。

```go
// 类型推导：无需声明类型
name := "Go语言"     // string
age := 15            // int
version := 1.22      // float64
```

## 原生并发：goroutine

goroutine 是 Go 最强大的特性。你可以轻松创建数万个并发任务，而系统开销几乎为零：

```go
// 关键字 go 启动一个 goroutine
go func() {
    fmt.Println("我正在并发执行")
}()
```

## channel：协程间的通信桥梁

channel 让协程之间的数据传递变得安全简单，无需锁机制即可实现并发通信：

```go
ch := make(chan string)
go func() { ch <- "数据" }()
msg := <-ch
```

## 其他显著特点

- **编译极快** — 整个项目秒级编译，媲美增量编译的脚本语言
- **垃圾回收** — 自动内存管理，告别手动内存分配
- **组合优于继承** — 用结构体嵌套和接口组合替代传统继承
- **标准库丰富** — net/http、encoding/json、os 等开箱即用
- **部署简单** — 编译成单个可执行文件，无依赖

## 设计哲学

Go 偏好显式而非隐式，简洁而非复杂。正如 Rob Pike 所说："Go 的设计就是让程序员少做蠢事。"
