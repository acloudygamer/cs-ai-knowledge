# Go 学习路线

## 学习阶段规划

### 第一阶段：入门（1-2周）

- 掌握基础语法：变量、数据类型、控制流程
- 理解函数的多返回值和错误处理
- 学会使用标准库（fmt、os、net/http）

### 第二阶段：进阶（2-4周）

- **goroutine 与 channel** — Go 并发的核心，深入理解CSP模型
- 理解接口与组合设计
- 掌握 defer/panic/recover 机制

### 第三阶段：实战（持续）

- 参与开源项目或构建自己的应用
- 学习常用框架（Gin、Echo）
- 了解 Go 最佳实践和性能调优

## 重点深入：goroutine 与 channel

这是 Go 区别于其他语言的核心，也是面试中最常考察的内容：

- **goroutine** — 轻量级线程，由 Go 运行时管理
- **channel** — 协程间通信的管道，有缓冲/无缓冲之分
- **select** — 多路复用，同时等待多个 channel
- **context** — 取消信号和超时控制

## 实践建议

1. **每天写代码** — Go 是实践性很强的语言，看十遍不如写一遍
2. **从读源码开始** — 阅读 Go 标准库源码是提升最快的方式之一
3. **参与社区** — GitHub、Go Forum、秦朋的公众号都是好去处
4. **构建项目** — 学到一定程度后，尝试用 Go 重写你熟悉的工具
