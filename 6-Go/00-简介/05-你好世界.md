# 你好，世界！

让我们编写第一个 Go 程序。创建文件 `hello.go`，输入以下代码：

```go
package main

import "fmt"

func main() {
    fmt.Println("你好，世界！")
}
```

现在运行它：

```bash
go run hello.go
# 输出：你好，世界！
```

## 代码逐行解释

### `package main`

每个 Go 文件都属于一个包（package）。`package main` 是特殊包名，表示这是一个可执行程序入口。Go 编译器会把 `package main` 的文件编译成可执行文件。

> 为什么是 main？因为程序需要一个入口点，Go 规定这个入口必须叫 main。

### `import "fmt"`

`import` 关键字用于导入其他包。`fmt` 是 Go 标准库中的格式化 I/O 包，提供了 `Println`、`Printf` 等打印函数。

如果你要用其他功能，就需要导入对应的包。例如，网络请求用 `net/http`，JSON 处理用 `encoding/json`。

### `func main()`

`func` 是定义函数的关键字。`main` 是特殊函数名——它是程序执行的起点。当运行程序时，Go 会自动调用 `main` 函数。

一个程序只能有一个 `main` 函数，而且必须放在 `package main` 的文件中。

### `fmt.Println("你好，世界！")`

`fmt.Println` 调用 fmt 包的 Println 函数，在控制台打印一行文字并换行。

## 扩展练习

试试修改代码，输出其他内容：

```go
package main

import "fmt"

func main() {
    name := "Go语言"
    fmt.Println("欢迎学习", name)
    fmt.Println("版本：", 1.22)
}
```

你能猜到输出是什么吗？运行看看！
