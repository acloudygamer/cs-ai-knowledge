# 贪心

## 核心思想

- 局部最优选择 → 全局最优
- 只考虑当前状态的最优解
- 不是所有问题都可用贪心
- **证明**：交换论证法、反证法

## 区间调度

```python
# 无重叠区间（贪心选择最早结束）
def erase_overlap_intervals(intervals):
    if not intervals:
        return 0
    intervals.sort(key=lambda x: x[1])
    end = intervals[0][1]
    count = 0
    for i in range(1, len(intervals)):
        if intervals[i][0] < end:
            count += 1
        else:
            end = intervals[i][1]
    return count

# 跳跃游戏
def can_jump(nums):
    furthest = 0
    for i in range(len(nums)):
        if i > furthest:
            return False
        furthest = max(furthest, i + nums[i])
    return True
```

## 哈夫曼编码思路

```python
import heapq

def huffman_merge(lengths):
    # 合并文件（类似哈夫曼编码）
    heap = lengths[:]
    heapq.heapify(heap)
    cost = 0
    while len(heap) > 1:
        a = heapq.heappop(heap)
        b = heapq.heappop(heap)
        cost += a + b
        heapq.heappush(heap, a + b)
    return cost

### 饼干分配

```python
# 分配尽可能多的孩子
def find_content_children(g, s):
    g.sort()  # 孩子需求
    s.sort()  # 饼干大小
    gi = si = 0
    while gi < len(g) and si < len(s):
        if s[si] >= g[gi]:
            gi += 1  # 满足一个孩子
        si += 1      # 饼干只能用一次
    return gi
```

### 船救人问题

```python
# 最少船数（每船最多2人，限制体重）
def num_rescue_boats(people, limit):
    people.sort()
    left, right = 0, len(people) - 1
    boats = 0
    while left <= right:
        if people[left] + people[right] <= limit:
            left += 1  # 尝试让最轻的人和最重的人同船
        right -= 1     # 最重的人一定上船
        boats += 1
    return boats
```

## 加油站问题

```python
# 环形路线加油站（判断能否绕一圈）
def can_complete_circuit(gas, cost):
    total_tank = 0
    curr_tank = 0
    start = 0

    for i in range(len(gas)):
        diff = gas[i] - cost[i]
        total_tank += diff
        curr_tank += diff
        if curr_tank < 0:
            start = i + 1      # 从下一个站点开始
            curr_tank = 0

    return start if total_tank >= 0 else -1

# 加油站的贪心正确性：
# 若从i到j油量变为负，则中间任何点都不可能到达j
# 所以直接跳过[i, j]从j+1重新开始
```

```python
# 判断能否用magazine的字符构成ransom
def can_construct(ransom_note, magazine):
    from collections import Counter
    return Counter(ransom_note) <= Counter(magazine)

# 贪心：优先使用magazine中较少的字符
def can_construct_greedy(ransom_note, magazine):
    ransom_count = {}
    for ch in ransom_note:
        ransom_count[ch] = ransom_count.get(ch, 0) + 1

    for ch in magazine:
        if ch in ransom_count and ransom_count[ch] > 0:
            ransom_count[ch] -= 1
            if ransom_count[ch] == 0:
                del ransom_count[ch]
            if not ransom_count:
                return True
    return len(ransom_count) == 0
```

## 柠檬水找零

```python
def lemonade_change(bills):
    five, ten = 0, 0
    for bill in bills:
        if bill == 5:
            five += 1
        elif bill == 10:
            if five == 0:
                return False
            five -= 1
            ten += 1
        else:  # bill == 20
            # 优先用10+5找零（保留5美元给其他顾客）
            if ten > 0 and five > 0:
                ten -= 1
                five -= 1
            elif five >= 3:
                five -= 3
            else:
                return False
    return True
```

## 单调递增数字

```python
# 找到小于等于N的最大单调递增数
def monotone_increasing_digits(N):
    digits = list(str(N))
    marker = len(digits)  # 标记需要修改的位置

    # 从右向左找到第一个递减的位置
    for i in range(len(digits) - 1, 0, -1):
        if digits[i - 1] > digits[i]:
            marker = i
            digits[i - 1] = str(int(digits[i - 1]) - 1)

    # 将marker之后的所有位都设为9
    for i in range(marker, len(digits)):
        digits[i] = '9'

    return int(''.join(digits))
```

## 贪心算法分类

| 问题 | 贪心策略 | 正确性条件 |
|------|----------|------------|
| 区间调度 | 选择最早结束 | 无重叠 |
| 加油站 | 从可支撑点重新开始 | 总油量>=总消耗 |
| 饼干分配 | 最小饼干满足最小需求 | 任意匹配 |
| 船救人 | 最重+最轻配对 | 体重限制=2 |
| 找零钱 | 优先用大面额 | 货币系统正规 |
| Huffman编码 | 小堆合并 | 前缀码 |

## 贪心 vs DP vs 回溯

| 问题类型 | 贪心 | 动态规划 | 回溯 |
|----------|------|----------|------|
| 最优子结构 | 有 | 有 | 可能有 |
| 子问题重叠 | 否 | 是 | 是 |
| 局部最优=全局最优 | 是 | 否 | 否 |
| 复杂度 | 通常O(n) | O(n^2)+ | 指数级 |
| 适用场景 | 分配、区间 | 计数、最值 | 枚举、组合 |
