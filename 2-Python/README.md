# Python

最适合入门的人工智能语言，简洁易读的语法让它成为全球最受欢迎的编程语言之一。

## 核心特点

- **简洁易读** - 缩进语法，代码即文档
- **动态类型** - 无需声明，灵活高效
- **生态丰富** - 「自带电池」，百万级第三方库
- **跨平台** - Windows / macOS / Linux 一处编写多处运行
- **AI 首选** - TensorFlow、PyTorch 等主流框架原生支持

## 适用场景

`Web开发` `数据分析` `机器学习` `自动化脚本` `网络爬虫` `AI应用`

## 学习路线

基础语法 → 数据结构 → 函数与模块 → 项目实践
