# 语言特点

## 动态类型

Python 是**动态类型**语言，变量无需声明类型，解释器会在运行时自动推断。这让代码编写更快捷，但也要注意类型错误。

```python
x = 10          # 整数
x = "hello"     # 字符串，合法但可能不是你的本意
```

## 解释型语言

Python 是**解释型**语言，代码无需编译，直接由解释器逐行执行。这使得开发周期短、调试方便，但运行速度相对编译型语言（如 C、Go）较慢。

## 简洁易读

Python 使用**缩进语法**表达代码块结构，避免了花括号或 begin/end 的视觉噪音。同一代码块的所有语句必须保持相同缩进。

```python
if True:
    print("缩进很重要")  # 这行属于 if 内部
    if True:
        print("嵌套的缩进")  # 更深的缩进层级
else:
    print("这行不会执行")  # 与 if 同级
```

## "Batteries Included" 哲学

Python 倡导「自带电池」，标准库非常丰富。安装 Python 后即可使用：

- `json` - JSON 处理
- `datetime` - 日期时间
- `re` - 正则表达式
- `math` - 数学运算
- `urllib` - 网络请求
- `sqlite3` - 数据库

## 丰富的生态

除了标准库，PyPI（Python Package Index）上有超过 40 万个第三方包，满足各种需求：

| 领域 | 代表库 |
|------|--------|
| Web开发 | Django, Flask, FastAPI |
| 数据科学 | NumPy, Pandas, Matplotlib |
| 机器学习 | TensorFlow, PyTorch, scikit-learn |
| 自动化 | Selenium, PyAutoGUI |

## 跨平台

Python 天然支持 Windows、macOS、Linux，一次编写多处运行。
