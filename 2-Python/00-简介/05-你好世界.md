# 你好世界

让我们从最简单的程序开始。

## 代码

```python
print("Hello, World!")
```

保存为 `hello.py` 后运行：

```bash
python hello.py
```

输出结果：

```
Hello, World!
```

## 逐行解释

### 第 1 行：`print("Hello, World!")`

| 组成部分 | 含义 |
|---------|------|
| `print` | Python 内置函数，用于输出内容到屏幕 |
| `(` | 函数调用的开始 |
| `"Hello, World!"` | 字符串，用引号包围的文本 |
| `)` | 函数调用的结束 |

### 为什么是引号？

引号告诉 Python 这是**字符串**——一串字符，而不是变量名或代码。如果不加引号：

```python
print(Hello, World!)  # 这会报错：NameError
```

Python 会把 Hello 当作变量，试图查找名为 Hello 的变量，而这个变量不存在。

### 为什么用 print？

在 Python 交互式环境中直接输入表达式会显示结果：

```python
>>> "Hello, World!"
'Hello, World!'
```

但在脚本文件中运行，必须用 `print()` 才能看到输出。`print` 函数还会自动换行，每次调用输出结束后会换到新的一行。

## 扩展练习
