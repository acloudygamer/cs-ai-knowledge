# 学习路线

## 推荐学习顺序

### 第一阶段：基础入门（1-2 周）

1. **语法基础** - 变量、数据类型、运算符
2. **控制流程** - if 条件、for/while 循环
3. **数据结构** - 列表、元组、字典、集合

目标是能读懂简单程序，能写 20-50 行的脚本。

### 第二阶段：函数与模块（1-2 周）

1. **函数定义** - 参数、返回值、作用域
2. **模块导入** - 标准库使用、第三方库安装
3. **文件操作** - 读写文本文件

这一阶段要多动手练习，尝试写一些文件处理小工具。

### 第三阶段：项目实践（持续）

选择一个感兴趣的方向深入：

- **数据分析方向** - NumPy、Pandas、Matplotlib
- **Web 开发方向** - Flask 或 Django
- **自动化脚本** - 文件处理、Excel 操作
- **爬虫实战** - requests、BeautifulSoup

通过项目巩固知识，遇到问题再回头查文档。

- B 站「小甲鱼 Python 教程」- 经典中文视频

## 学习建议

1. **每天写代码** - 哪怕只有 30 分钟
2. **不懂就查文档** - 官方文档是最好的教程
3. **多读别人的代码** - GitHub 上有大量优质开源项目
4. **不要怕踩坑** - 错误是最好的老师

坚持学习 2-3 个月后，你就能用 Python 完成日常自动化任务和小型项目了。
