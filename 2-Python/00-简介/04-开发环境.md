# 开发环境

## Python 安装

### Windows

1. 访问 **python.org** 下载安装包
2. 运行安装程序，**务必勾选「Add Python to PATH」**
3. 打开命令提示符，输入 `python --version` 验证

### macOS

macOS 通常预装 Python，但版本较旧。建议使用 Homebrew 安装：

```bash
brew install python3
python3 --version
```

### Linux

大多数发行版已自带 Python。Ubuntu/Debian 可用：

```bash
sudo apt update
sudo apt install python3 python3-pip
```

## Python Launcher (Windows)

Windows 用户建议安装 **py Launcher**，它可以方便地管理多个 Python 版本：

```bash
py -3.11    # 运行 Python 3.11
py -3.12    # 运行 Python 3.12
py -0       # 列出所有已安装版本
```

## IDE 选择

| 编辑器 | 特点 | 适合人群 |
|--------|------|----------|
| **VS Code** | 轻量、插件丰富 | 所有人群 |
| **PyCharm** | 专业、强大、收费版功能更全 | 专业开发者 |
| **Jupyter Notebook** | 交互式编程 | 数据科学/实验 |
| **Thonny** | 极简、内置新手调试器 | 完全初学者 |

推荐初学者从 **VS Code + Python 插件** 开始，它免费、轻量且功能完善。

## 安装验证

安装完成后，打开终端/命令提示符验证：

```bash
python --version    # 应显示 Python 3.x.x
pip --version       # 应显示 pip 版本信息
```

如果遇到「不是内部命令」错误，重启终端或检查 PATH 环境变量。
