# 应用场景

## 人工智能与机器学习

Python 是 AI 领域的主流语言，几乎所有主流深度学习框架都提供 Python API：

- **TensorFlow** - 谷歌开源的机器学习框架
- **PyTorch** - Facebook 主导的深度学习框架
- **scikit-learn** - 经典的机器学习库
- **Keras** - 高级神经网络 API

AI 方向的课程、教程、论文复现几乎都以 Python 为首选。

## Web 开发

Python Web 框架生态成熟，从微服务到大型应用都能胜任：

- **Django** - 全功能框架，内置 ORM、管理后台、认证
- **Flask** - 轻量级微框架，灵活度高
- **FastAPI** - 现代高性能框架，自动生成 API 文档

## 数据分析与科学计算

数据科学是 Python 最强势的领域之一：

- **NumPy** - 高性能数值计算基础库
- **Pandas** - 数据分析利器，表格处理
- **Matplotlib / Seaborn** - 数据可视化
- **SciPy** - 科学计算工具集

## 自动化与脚本

Python 是编写自动化脚本的首选语言：

- 文件批处理
- Excel/Word 操作
- 浏览器自动化（配合 Selenium）
- 定时任务管理

## 网络爬虫

Python 拥有最成熟的爬虫生态：

```python
import requests
from bs4 import BeautifulSoup

response = requests.get("https://example.com")
soup = BeautifulSoup(response.text, "html.parser")
print(soup.title.text)
```

常用库包括 `requests`、`BeautifulSoup`、`Scrapy`、`Selenium` 等。

## 其他领域

- 游戏开发（PyGame）

- 网络编程
- 云计算（AWS Lambda 支持 Python）
