# 语言简介

## 历史背景

Python 由荷兰程序员 **Guido van Rossum**（吉多·范罗苏姆）于 **1991年** 创立。Guido 在1989年圣诞节期间着手开发这门语言，灵感来源于他对ABC语言的参与经验以及对 Amoeba 操作系统开发的反思。

## 命名由来

Python 的名字并非来自蟒蛇，而是来自英国喜剧团 Monty Python。Guido 是 Monty Python 飞行马戏团的忠实粉丝，希望这门语言能给开发者带来乐趣。

## 版本演进

| 版本 | 发布年份 | 状态 |
|------|----------|------|
| Python 1.0 | 1991 | 已停止维护 |
| Python 2.0 | 2000 | 已停止维护（2020年终止） |
| Python 3.0 | 2008 | 当前主流版本 |

**Python 2 → 3 的迁移**是历史上重要的变革。Python 3 修正了许多设计缺陷（如 print 语句改为 print 函数、字符串默认 Unicode 等），但 breaking change 导致大量旧代码需要改写。如今新项目应始终使用 Python 3。

## 设计哲学

Python 的设计理念浓缩在「PEP 20 - The Zen of Python」中，可用 `import this` 查看。其中核心原则包括：

- **简洁优于复杂**
- **明了优于隐晦**
- **可读性很重要**

这种设计哲学使 Python 成为最适合入门的编程语言之一。
