# 脚本编程

## 概念

**Shell脚本**是用Shell语言编写的文本文件，包含一系列命令，供Shell解释执行。

```bash
#!/bin/bash
# shebang: 告诉系统用哪个解释器执行

echo "Hello, World!"
```

## 关系

**关键连接**：
- 脚本 → **Shell**：脚本本质是Shell命令的文本化
- 脚本 → **文件系统**：脚本是磁盘上的文件，需要有执行权限
- 脚本 → **环境变量**：脚本可以读取和设置环境变量
- 自动化 → **脚本**：cron、systemd等调用脚本实现自动化

## 基础语法

### 变量
```bash
# 定义和赋值（等号两边不能有空格）
name="Alice"
age=25

# 使用变量
echo $name
echo "${name}'s age is ${age}"

# 只读变量
readonly PI=3.14159

# 删除变量
unset name
```

### 特殊变量
```bash
$0    # 脚本名
$1    # 第一个参数
$2    # 第二个参数
$@    # 所有参数
$#    # 参数个数
$$    # 当前进程ID
$?    # 上一条命令的退出码
```

### 引号
```bash
name="Alice"

echo $name          # Alice
echo "$name"        # Alice
echo '$name'        # $name (单引号不解析变量)
```

## 条件判断

### test 命令
```bash
# 文件测试
[ -f "file.txt" ]   # 文件存在且是普通文件
[ -d "dir" ]        # 目录存在
[ -r "file" ]       # 文件可读
[ -w "file" ]       # 文件可写
[ -x "file" ]       # 文件可执行

# 字符串测试
[ -z "$str" ]       # 字符串为空
[ "$a" = "$b" ]     # 字符串相等
[ "$a" != "$b" ]    # 字符串不等

# 数字测试
[ $a -eq $b ]       # 相等
[ $a -gt $b ]       # 大于
[ $a -lt $b ]       # 小于
```

### if 语句
```bash
if [ -f "file.txt" ]; then
    echo "File exists"
elif [ -d "dir" ]; then
    echo "Directory exists"
else
    echo "Not found"
fi

# 单行写法
[ -f "file.txt" ] && echo "exists"
```

## 循环

### for 循环
```bash
# 列表循环
for i in 1 2 3 4 5; do
    echo "Number: $i"
done

# C风格
for ((i=0; i<5; i++)); do
    echo "i = $i"
done

# 遍历文件
for f in *.txt; do
    echo "File: $f"
done

# 遍历命令输出
for line in $(cat file.txt); do
    echo $line
done
```

### while 循环
```bash
count=0
while [ $count -lt 5 ]; do
    echo $count
    count=$((count + 1))
done

# 读取文件
while read line; do
    echo $line
done < file.txt
```

## 函数

```bash
# 定义函数
greet() {
    echo "Hello, $1!"
    return 0
}

# 调用函数
greet "Alice"

# 带返回值的函数
get_sum() {
    echo $(( $1 + $2 ))
}

sum=$(get_sum 10 20)
```

## 数组

```bash
# 定义数组
arr=(one two three four)

# 访问元素
echo ${arr[0]}      # one
echo ${arr[@]}      # 全部元素
echo ${#arr[@]}     # 数组长度

# 关联数组 (bash 4+)
declare -A person
person["name"]="Alice"
person["age"]=25
```

## 输入输出

### 读取用户输入
```bash
echo "Enter your name:"
read name
echo "Hello, $name"

# 带提示的读取
read -p "Enter your age: " age

# 密码输入（不显示）
read -sp "Password: " password
```

### 格式化输出
```bash
printf "Name: %s, Age: %d\n" "Alice" 25
printf "Pi: %.2f\n" 3.14159
```

## 实战脚本

### 备份脚本
```bash
#!/bin/bash
# backup.sh - 备份目录

BACKUP_DIR="/backup"
SOURCE_DIR=$1

if [ -z "$SOURCE_DIR" ]; then
    echo "Usage: $0 <directory>"
    exit 1
fi

timestamp=$(date +%Y%m%d_%H%M%S)
backup_file="${BACKUP_DIR}/backup_${timestamp}.tar.gz"

tar -czf "$backup_file" "$SOURCE_DIR"
if [ $? -eq 0 ]; then
    echo "Backup created: $backup_file"
else
    echo "Backup failed!"
    exit 1
fi
```

### 日志分析
```bash
#!/bin/bash
# analyze_log.sh - 分析日志错误

LOG_FILE=${1:-/var/log/syslog}

echo "=== Error Summary ==="
grep -c "error" "$LOG_FILE"
echo "total errors"

echo "=== Recent Errors ==="
tail -10 "$LOG_FILE" | grep -i error
```

## 调试

```bash
# 选项
set -x    # 打印每条命令及参数
set -e    # 遇错误立即退出
set -u    # 使用未定义变量时报错
set -o pipefail  # 管道中任何一个失败则整体失败

# 示例
#!/bin/bash
set -xeuo pipefail

# 或者在命令行调试
bash -x script.sh
```

## 面试要点

1. **shebang**：`#!/bin/bash`的作用
2. **变量**：定义、使用、只读变量、环境变量
3. **条件判断**：`[]`和`[[]]`的区别
4. **循环**：for/while的多种写法
5. **函数**：定义、参数、返回值
6. **数组**：普通数组和关联数组
7. **调试**：`set -x`跟踪执行，`set -e`遇错退出
