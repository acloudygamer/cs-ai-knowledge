# 脚本编程

## 概念

**Shell脚本**是用Shell语言编写的文本文件，包含一系列命令，供Shell解释执行。

```bash
#!/bin/bash
# shebang: 告诉系统用哪个解释器执行

echo "Hello, World!"
```

## 关系

**关键连接**：
- 脚本 → **Shell**：脚本本质是Shell命令的文本化
- 脚本 → **文件系统**：脚本是磁盘上的文件，需要有执行权限
- 脚本 → **环境变量**：脚本可以读取和设置环境变量
- 自动化 → **脚本**：cron、systemd等调用脚本实现自动化

## 基础语法

### 变量
```bash
# 定义和赋值（等号两边不能有空格）
name="Alice"
age=25

# 使用变量
echo $name
echo "${name}'s age is ${age}"

# 只读变量
readonly PI=3.14159

# 删除变量
unset name
```

### 特殊变量
```bash
$0    # 脚本名
$1    # 第一个参数
$2    # 第二个参数
$@    # 所有参数
$#    # 参数个数
$$    # 当前进程ID
$?    # 上一条命令的退出码
```

### 引号
```bash
name="Alice"

echo $name          # Alice
echo "$name"        # Alice
echo '$name'        # $name (单引号不解析变量)
```

## 条件判断

### test 命令
```bash
# 文件测试
[ -f "file.txt" ]   # 文件存在且是普通文件
[ -d "dir" ]        # 目录存在
[ -r "file" ]       # 文件可读
[ -w "file" ]       # 文件可写
[ -x "file" ]       # 文件可执行

# 字符串测试
[ -z "$str" ]       # 字符串为空
[ "$a" = "$b" ]     # 字符串相等
[ "$a" != "$b" ]    # 字符串不等

# 数字测试
[ $a -eq $b ]       # 相等
[ $a -gt $b ]       # 大于
[ $a -lt $b ]       # 小于
```

### if 语句
```bash
if [ -f "file.txt" ]; then
    echo "File exists"
elif [ -d "dir" ]; then
    echo "Directory exists"
else
    echo "Not found"
fi

# 单行写法
[ -f "file.txt" ] && echo "exists"
```

## 循环

### for 循环
```bash
# 列表循环
for i in 1 2 3 4 5; do
    echo "Number: $i"
done

# C风格
for ((i=0; i<5; i++)); do
    echo "i = $i"
done

# 遍历文件
for f in *.txt; do
    echo "File: $f"
done

# 遍历命令输出
for line in $(cat file.txt); do
    echo $line
done
```

### while 循环
```bash
count=0
while [ $count -lt 5 ]; do
    echo $count
    count=$((count + 1))
done

# 读取文件
while read line; do
    echo $line
done < file.txt
```

## 函数

```bash
# 定义函数
greet() {
    echo "Hello, $1!"
    return 0
}

# 调用函数
greet "Alice"

# 带返回值的函数
get_sum() {
    echo $(( $1 + $2 ))
}

sum=$(get_sum 10 20)
```

## 数组

## 实战脚本

### 备份脚本
```bash
#!/bin/bash
# backup.sh - 备份目录

BACKUP_DIR="/backup"
SOURCE_DIR=$1

if [ -z "$SOURCE_DIR" ]; then
    echo "Usage: $0 <directory>"
    exit 1
fi

timestamp=$(date +%Y%m%d_%H%M%S)
backup_file="${BACKUP_DIR}/backup_${timestamp}.tar.gz"

tar -czf "$backup_file" "$SOURCE_DIR"
if [ $? -eq 0 ]; then
    echo "Backup created: $backup_file"
else
    echo "Backup failed!"
    exit 1
fi
```

### 日志分析
```bash
#!/bin/bash
# analyze_log.sh - 分析日志错误

LOG_FILE=${1:-/var/log/syslog}

echo "=== Error Summary ==="
grep -c "error" "$LOG_FILE"
echo "total errors"

echo "=== Recent Errors ==="
tail -10 "$LOG_FILE" | grep -i error
```

## 字符串处理

```bash
# 字符串长度
str="Hello World"
echo ${#str}              # 11

# 子字符串
echo ${str:0:5}           # Hello
echo ${str: -5}           # World（注意空格）

# 字符串替换
echo ${str/Hello/Hi}      # Hi World
echo ${str//o/O}          # HellO WOrld（全局）
echo ${str/#Hello/Hi}     # Hi World（前缀）
echo ${str/%World/Universe} # Hello Universe（后缀）

# 字符串去空格
text="  hello  "
echo "${text}"            # "  hello  "
echo "${text// }"         # "hello"

# 字符串转大写/小写 (bash 4+)
echo "${str^^}"           # HELLO WORLD
echo "${str,,}"           # hello world
```

## 进程替换与命令替换

```bash
# 命令替换：$(...) 或反引号
now=$(date)
files=$(ls -la)

# 进程替换：<(...) 和 >(...)
# 将命令输出当作文件
diff <(ls /dir1) <(ls /dir2)

# 同时读写
# >(cmd) 将输出通过管道写入cmd
echo "data" | tee >(process1) >(process2) >/dev/null

# 用while读命令输出（更高效）
while IFS= read -r line; do
    echo "Line: $line"
done < <(grep "pattern" file.txt)
# 注意：< <(...) 而不是 < $(...)
```

##trap 信号捕获

脚本运行时捕获信号处理：

```bash
#!/bin/bash
# 捕获信号

trap 'echo "Ctrl+C detected"; exit 1' INT
trap 'echo "脚本退出"; cleanup' EXIT
trap 'echo "收到HUP信号"' HUP

# cleanup函数
cleanup() {
    echo "清理临时文件..."
    rm -f /tmp/script_$$.tmp
}

# 任何错误都退出
set -e
```

## 高级模式

### set 选项详解

```bash
#!/bin/bash
set -e    # 遇错误立即退出（但要注意命令在管道中的情况）
set -u    # 使用未定义变量时报错
set -x    # 打印每条命令及参数（调试）
set -o pipefail  # 管道中任何一个失败则整体失败
set -n    # 语法检查但不执行（用于调试）
set -v    # 打印原始输入行

# 组合使用
set -euo pipefail

# 或者在脚本开头
#!/bin/bash -euxo pipefail

# 在函数内使用local
set -u
myfunc() {
    local var=${1:-}  # 使用默认值防止-u报错
}
```

### 错误处理模式

```bash
# 错误处理函数
error_exit() {
    echo "Error: $1" >&2
    exit 1
}

# 使用
[ -f "$file" ] || error_exit "File not found: $file"

# 或带条件
if ! cd "$dir"; then
    error_exit "Cannot change to directory: $dir"
fi

# 命令成功/失败检查
command && echo "Success" || echo "Failed"
```

## 实用模式

### getopts 处理选项

```bash
#!/bin/bash
# 处理命令行选项

usage() {
    echo "Usage: $0 [-f file] [-v] [-n name]"
    exit 1
}

# -f: 需要参数，-v: 标志，-n: 需要参数
while getopts "f:vn:" opt; do
    case $opt in
        f) file="$OPTARG" ;;
        v) verbose=true ;;
        n) name="$OPTARG" ;;
        *) usage ;;
    esac
done
shift $((OPTIND-1))

echo "File: $file, Verbose: $verbose, Name: $name"
```
