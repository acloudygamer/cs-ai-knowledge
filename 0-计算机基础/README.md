# 计算机基础

本部分不是工具手册，而是**概念关系图谱**——帮你理解计算机各层如何连接。

## 概念层次

```
┌─────────────────────────────────────────────────────────────┐
│                     用户/应用程序                           │
├─────────────────────────────────────────────────────────────┤
│  前端/后端/UI/测试/算法 → 软件工程概念层                    │
├─────────────────────────────────────────────────────────────┤
│  编程语言: Python/C++/Java/JS/Go                            │
│    ↓ 编译器/解释器/运行时                                  │
├─────────────────────────────────────────────────────────────┤
│  操作系统: Linux/Windows/Android/iOS                        │
│    ↓ 内核 + 系统调用                                        │
├─────────────────────────────────────────────────────────────┤
│  Shell: bash/zsh/PowerShell/伪终端                          │
├─────────────────────────────────────────────────────────────┤
│                     硬件: CPU/内存/存储/网络                 │
└─────────────────────────────────────────────────────────────┘
```

## 关键连接关系

| 关系 | 说明 |
|------|------|
| **终端 → Shell → 内核 → 硬件** | 用户如何与系统交互 |
| **编译器/解释器 ↔ 运行时 ↔ 操作系统** | 代码如何被执行 |
| **移动OS架构** | Android/iOS 如何在内核上构建 |
| **Node.js 运行时栈** | V8 + libuv + 事件循环 |
| **DNS解析流程** | 浏览器 → 递归DNS → 根/TLD/权威服务器 → IP返回 |
| **TCP/IP协议栈** | HTTP → TCP → IP → Ethernet，数据层层封装 |

## 学习路径

1. **先硬件**：了解计算机由什么组成
2. **再系统软件**：理解OS如何管理硬件
3. **后编程基础**：掌握代码如何被编译/执行
4. **交互层**：学会使用Shell与系统交互
5. **工程概念**：理解软件开发的整体图景

建议按顺序学习，但各部分之间有交叉引用，可随时跳转。
