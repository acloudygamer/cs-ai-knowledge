# 硬件组成

## 概念

计算机硬件由**CPU、内存、存储、输入/输出设备**四大核心组件构成，通过总线互联。

```
┌─────────────────────────────────────┐
│              用户/程序              │
└────────────────┬──────────────────┘
                 │ I/O设备交互
┌────────────────▼──────────────────┐
│           I/O 设备                │
│  (键盘、显示器、网卡、磁盘)        │
└────────────────┬──────────────────┘
                 │
┌────────────────▼──────────────────┐
│         存储 (磁盘/SSD)           │
└────────────────┬──────────────────┘
                 │
┌────────────────▼──────────────────┐
│           内存 (RAM)              │
└────────────────┬──────────────────┘
                 │
┌────────────────▼──────────────────┐
│            CPU                    │
│    (控制单元 + 运算单元 + 寄存器)  │
└─────────────────────────────────────┘
```

## 关系

**关键连接**：
- CPU ↔ **内存**：CPU直接访问内存，速度快
- 内存 → **存储**：存储是永久性内存，内存是临时性
- I/O设备 → **CPU**：通过中断通知CPU
- 所有组件 → **主板/总线**：物理连接通道

## CPU (中央处理器)

CPU是计算机的大脑，负责执行指令：

```python
# 程序执行流程（简化）
instructions = load_program_from_memory()
for instruction in instructions:
    pc = execute(instruction)  # PC寄存器指向下一条指令
```

**CPU核心指标**：

| 指标 | 说明 |
|------|------|
| 主频 | CPU时钟频率，GHz为单位，越高越快 |
| 核心数 | 可并行处理的核心数 |
| 线程数 | 超线程技术，一个核心可处理多个线程 |
| 缓存 | L1/L2/L3，越大越好 |

## 内存 (RAM)

内存用于存储**运行时数据**，CPU可直接访问：

```c
// 内存布局示意
int global_var = 10;        // .data 段（全局/静态）
static int static_var = 20; // .data 段

int main() {
    int local_var = 30;     // 栈上（局部变量）
    int* heap_ptr = malloc(sizeof(int));  // 堆上（动态分配）
    free(heap_ptr);
    return 0;
}
```

**内存层级**：

| 层级 | 速度 | 容量 | 成本 |
|------|------|------|------|
| 寄存器 | 最快 (0周期) | 极小 | 最高 |
| L1缓存 | 快 (4周期) | 32KB-64KB | 高 |
| L2缓存 | 中 (12周期) | 256KB-1MB | 中 |
| L3缓存 | 较慢 (40周期) | 4MB-64MB | 低 |
| 内存 | 慢 (100+周期) | GB级 | 最低 |

## 存储 (磁盘/SSD)

存储用于**永久保存**数据：

```bash
# 查看磁盘信息 (Linux)
lsblk
df -h
fdisk -l

# 查看磁盘信息 (Windows)
wmic diskdrive get status,size,model
```

| 存储类型 | 原理 | 速度 | 容量 | 寿命 |
|----------|------|------|------|------|
| HDD | 机械旋转+磁头 | 慢 (100MB/s) | 大 (TB级) | 机械损耗 |
| SSD | Flash闪存 | 快 (500MB/s) | 中 (TB级) | 写入次数限制 |
| NVMe SSD | PCIe接口 | 极快 (GB/s) | 中 | 写入次数限制 |

## I/O 设备

输入/输出设备负责与外部世界交互：

| 设备类型 | 示例 | 数据方向 |
|----------|------|----------|
| 输入设备 | 键盘、鼠标、麦克风 | 外设 → CPU |
| 输出设备 | 显示器、扬声器 | CPU → 外设 |
| 存储设备 | 磁盘、U盘 | 双向 |
| 网络设备 | 网卡 | 双向 |

## 总线 (Bus)

总线是组件间数据传输的通道：

```python
# 总线类型示意
system_bus = {
    "数据总线": "传输实际数据（位宽：32/64位）",
    "地址总线": "指定内存地址（决定寻址空间）",
    "控制总线": "传输控制信号（读/写/中断）"
}
```

## 面试要点

1. **CPU执行指令流程**：取指→译码→执行→访存→写回
2. **内存层级**：寄存器→缓存→内存→存储的速度和容量权衡
3. **冯·诺依曼架构**：存储程序原理，代码和数据同等对待
4. **CPU缓存**：L1/L2/L3区别，缓存一致性
5. **HDD vs SSD**：原理区别，适用场景
