# 算法工程

## 概念

**算法工程**是将算法应用于实际问题的实践，关注算法在工程中的复杂度、性能和实用性。

```
问题 → 算法选择 → 实现 → 优化 → 生产部署
```

## 关系

**关键连接**：
- 算法 → **复杂度分析**：时间/空间复杂度决定算法适用场景
- 算法 → **数据结构**：不同数据结构支持不同操作
- 算法 → **实际应用**：排序/搜索/图论在工程中的具体应用
- 算法 → **面试**：算法题是技术面试的核心
- 工程 → **算法**：不是最优解最好，而是最合适的解

## 复杂度分析

### 时间复杂度
```python
# O(1) - 常数时间
def get_first_element(arr):
    return arr[0]  # 不管数组多大，取第一个元素时间不变

# O(log n) - 对数时间
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# O(n) - 线性时间
def find_max(arr):
    max_val = arr[0]
    for val in arr:
        if val > max_val:
            max_val = val
    return max_val

# O(n log n) - 线性对数时间
# 排序算法如TimSort (Python sorted())

# O(n²) - 平方时间
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

### 复杂度等级
| 复杂度 | 名称 | 能处理的数据规模 |
|--------|------|------------------|
| O(1) | 常数 | 任意规模 |
| O(log n) | 对数 | 十亿级 |
| O(n) | 线性 | 百万级 |
| O(n log n) | 线性对数 | 十万级 |
| O(n²) | 平方 | 千级 |
| O(2ⁿ) | 指数 | 20-30 |

## 工程中的算法应用

### 搜索
```python
# 生产环境中的搜索
def search_products(query, products, limit=20):
    """电商商品搜索"""
    # 1. 预处理：分词
    tokens = query.lower().split()

    # 2. 过滤：标题匹配
    candidates = [
        p for p in products
        if any(token in p['title'].lower() for token in tokens)
    ]

    # 3. 排序：相关性 + 质量分数
    def relevance_score(product):
        title_match = sum(1 for t in tokens if t in product['title'].lower())
        return title_match / len(tokens) * product['rating']

    candidates.sort(key=relevance_score, reverse=True)

    return candidates[:limit]
```

### 缓存与LRU
```python
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)  # 最近访问
        return self.cache[key]

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)  # 删除最旧的
```

### 图算法在工程中的应用
```python
# 社交网络好友推荐（简化版）
from collections import defaultdict

def recommend_friends(user_id, friendships):
    """基于共同好友的好友推荐"""
    # 构建邻接表
    graph = defaultdict(set)
    for u, v in friendships:
        graph[u].add(v)
        graph[v].add(u)

    # 推荐：朋友的朋友（排除已好友）
    recommendations = defaultdict(int)
    for friend in graph[user_id]:
        for suggested in graph[friend]:
            if suggested != user_id and suggested not in graph[user_id]:
                recommendations[suggested] += 1

    # 按共同好友数排序
    return sorted(recommendations.items(), key=lambda x: -x[1])
```

## 实用算法思维

### 分治法
```python
# 归并排序
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

### 动态规划
```python
# 爬楼梯问题
def climb_stairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 优化：只用两个变量
def climb_stairs_optimized(n):
    if n <= 2:
        return n
    prev, curr = 1, 2
    for _ in range(3, n + 1):
        prev, curr = curr, prev + curr
    return curr
```

### 贪心算法
```python
# 活动选择问题
def activity_selection(activities):
    """选择最多不重叠的活动"""
    # 按结束时间排序
    sorted_activities = sorted(activities, key=lambda x: x[1])

    count = 0
    last_end = 0
    for start, end in sorted_activities:
        if start >= last_end:
            count += 1
            last_end = end

    return count
```
