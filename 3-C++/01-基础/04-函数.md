# 函数

## 默认参数

```cpp
void greet(std::string name = "World") {
    std::cout << "Hello, " << name << "!" << std::endl;
}

greet();           // Hello, World!
greet("Alice");    // Hello, Alice!
```

## 函数重载

```cpp
// 同名函数，不同参数
int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }
std::string add(std::string a, std::string b) { return a + b; }
```

## 内联函数

```cpp
inline int max(int a, int b) {
    return (a > b) ? a : b;
}
// 编译器可能将函数体直接展开，避免函数调用开销
```

## Lambda 表达式（C++11）

```cpp
#include <algorithm>

std::vector<int> nums = {5, 2, 8, 1, 9};

// 完整 lambda
auto is_even = [](int n) -> bool {
    return n % 2 == 0;
};

// 省略返回类型推导
auto square = [](int n) { return n * n; };

// 带捕获的 lambda
int threshold = 5;
auto greater_than = [threshold](int n) {
    return n > threshold;
};

// 使用 std::function
std::function<int(int, int)> add = [](int a, int b) { return a + b; };

// 在算法中使用
std::sort(nums.begin(), nums.end(), [](int a, int b) {
    return a > b;  // 降序
});

auto it = std::find_if(nums.begin(), nums.end(), is_even);
```

### Lambda 捕获模式

```cpp
int x = 10;
int y = 20;

// 按值捕获：复制当前值
[x, y]() { return x + y; }();

// 按引用捕获：保持与外部变量同步
[&x, &y]() { x = 100; return x + y; }();

// 混合捕获：x按值，y按引用
[x, &y]() { return x + y; }();

// 默认捕获模式
[=]() { return x + y; }();      // 默认按值捕获所有变量
[&]() { return x + y; }();      // 默认按引用捕获所有变量
[=, &x]() { return x + y; }();  // 默认按值，x按引用
[&, =x]() { return x + y; }();  // 默认按引用，x按值（C++17）

// 初始化捕获（C++14）：创建新变量
[z = x + y]() { return z; }();

// C++14 捕获表达式
auto ptr = std::make_unique<int>(42);
[wp = std::weak_ptr(ptr)]() { return wp.lock() != nullptr; };
```

### mutable Lambda

按值捕获的 lambda 默认不能修改捕获变量，使用 `mutable` 允许修改：

```cpp
int count = 0;

// 按值捕获，mutable 允许修改 count 的副本
auto counter = [count]() mutable {
    count++;
    return count;
};

counter();  // 返回 1
counter();  // 返回 2
count;     // 仍然是 0，原始变量不受影响
```

### Generic Lambda（C++14）

参数使用 `auto` 让 lambda 成为泛型：

```cpp
auto plus = [](auto a, auto b) { return a + b; };

plus(1, 2);           // int: 3
plus(1.5, 2.5);       // double: 4.0
plus(std::string("a"), std::string("b"));  // "ab"

// 与标准库算法结合
std::vector<int> nums = {1, 2, 3};
std::transform(nums.begin(), nums.end(), nums.begin(), [](auto n) {
    return n * 2;
});
```

### constexpr Lambda（C++17）

编译期计算的 lambda：

```cpp
constexpr auto add = [](auto a, auto b) constexpr {
    return a + b;
};

constexpr int result = add(3, 4);  // 编译时计算，result = 7
```

### Lambda 与 std::bind

Lambda 优于 std::bind 的场景：

```cpp
auto bound = std::bind(func, std::placeholders::_1, 100);

// 等价 lambda，更清晰
auto lambda = [](auto&& arg) { return func(arg, 100); };

// Lambda 优势：语法直观、可读性强、支持 mutable、捕获灵活
```

### IIFE（立即调用函数表达式）

```cpp
// C++14 开始支持初始化捕获结合 IIFE
int x = 10;
auto result = [x = x + 1]() { return x; }();  // result = 11

// 用于初始化复杂对象
auto [a, b] = [str = std::string("hello")]() {
    return std::pair<char, size_t>{str[0], str.size()};
}();
```

### Lambda 作为函数参数

```cpp
// 高阶函数：接收 lambda
template<typename Func>
void transform(std::vector<int>& vec, Func&& func) {
    for (auto& n : vec) {
        n = func(n);
    }
}

transform(nums, [](int n) { return n * 2; });

// 使用 std::function 指定类型
void process(std::function<bool(int)> predicate);
```

## 函数指针

```cpp
int (*operation)(int, int) = add;  // 指向 add 函数
int result = operation(3, 4);     // 7
```
