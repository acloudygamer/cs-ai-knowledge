# 变量与类型

## 静态类型与类型推导

- C++ 是静态类型语言，编译时确定类型
- `auto` 关键字自动推导类型（C++11）

```cpp
auto x = 42;          // int
auto y = 3.14;         // double
auto name = "hello";  // const char*
auto vec = std::vector<int>{1, 2, 3};  // std::vector<int>
```

## auto 与 decltype

### auto 类型推导

```cpp
auto a = 42;         // int
auto b = 3.14;        // double
auto c = 'c';         // char
auto d = std::string("test");  // std::string

// 注意：auto 会移除引用和 cv 限定符
int x = 10;
int& ref = x;
auto y = ref;         // int，不是 int&

// 使用 auto& 保留引用
auto& z = ref;        // int&

// 使用 auto&& 保留值类别（左值引用或右值引用）
auto&& r1 = x;        // int&
auto&& r2 = 10;       // int&&
```

### decltype 类型推导

decltype 推导但不求值，返回类型：

```cpp
int x = 10;
int& ref = x;
int* ptr = &x;

decltype(x) a = 5;    // int
decltype(ref) b = x;  // int&，必须初始化为左值
decltype(ptr) c = &x; // int*

// decltype((expr)) 双括号推导为引用
int i = 10;
decltype((i)) r1 = i;  // int&（表达式结果是引用）
decltype(i) r2 = i;    // int

// 用于函数返回类型
template<typename T, typename U>
auto add(T a, U b) -> decltype(a + b) {
    return a + b;
}
```

### decltype(auto)（C++14）

保留表达式的原始类型（包括引用和 cv 限定符）：

```cpp
int x = 10;
int& get_ref() { return x; }

decltype(auto) r1 = get_ref();  // int&，保留返回类型
decltype(auto) r2 = x;          // int，不会推导为 int&

// 错误示例：auto 会丢失引用
// auto r3 = get_ref();  // int，不是 int&
```

## decltype vs auto

```cpp
int x = 10;
const int cx = x;
int& rx = x;

auto a = rx;          // int，移除引用
auto b = cx;          // int，移除 const
decltype(rx) c = rx;  // int&，保留引用
decltype(cx) d = cx;  // const int，保留 const
```

## 类型转换

```cpp
int a = 10;
double b = static_cast<double>(a);    // 编译时转换

// 基类派生类转换（运行时检查）
Base* base_ptr = new Derived();
Derived* derived_ptr = dynamic_cast<Derived*>(base_ptr);

// const 移除
const_cast<int*>(&a);

// reinterpret 底层位模式重解释
reinterpret_cast<int*>(some_ptr);
```

### static_cast

编译时类型转换，用于明确定义的转换：

```cpp
double d = 3.14;
int i = static_cast<int>(d);  // 3，截断小数

void* p = &d;
double* dp = static_cast<double*>(p);  // 必须从 void* 转换回来

enum Color { Red, Green, Blue };
int c = static_cast<int>(Red);  // 枚举到整数
```

### dynamic_cast

运行时类型检查，安全向下转型：

```cpp
class Base { virtual ~Base() {} };
class Derived : public Base { void special() {} };

Base* base_ptr = new Derived();
Derived* derived_ptr = dynamic_cast<Derived*>(base_ptr);
if (derived_ptr) {
    derived_ptr->special();  // 安全使用
}

// 引用类型的 dynamic_cast（失败抛异常）
Derived& ref = dynamic_cast<Derived&>(*base_ptr);  // 成功返回引用，失败抛 bad_cast
```

### const_cast

移除或添加 const/volatile：

```cpp
const int ci = 10;
// int* p = &ci;  // 错误：不能从 const int* 赋值到 int*
int* p = const_cast<int*>(&ci);  // 移除 const

// 合理用法：通过非 const 路径调用函数
void mutate(std::string& s) { s[0] = 'X'; }
const std::string s = "hello";
const_cast<std::string&>(s)[0] = 'X';  // 调用后 s 实际被修改（未定义行为）
```

### reinterpret_cast

低层位重解释，不进行任何检查：

```cpp
int* ip = new int(42);
char* cp = reinterpret_cast<char*>(ip);  // 同一地址的不同类型解释

// 底层位操作
long long addr = reinterpret_cast<long long>(ip);

// 函数指针类型转换
typedef void (*FuncPtr)();
FuncPtr fp = reinterpret_cast<FuncPtr>(ip);  // 危险，仅示例
```

## 类型别名

```cpp
// typedef（C++98）
typedef unsigned int uint;
typedef std::vector<int> IntVec;
typedef int (*FuncPtr)(int, int);

// using 别名（C++11，更清晰）
using uint = unsigned int;
using IntVec = std::vector<int>;
using FuncPtr = int(*)(int, int);

// 模板别名（C++14）
template<typename T>
using Vec = std::vector<T>;
Vec<int> int_vec;
Vec<std::string> str_vec;
```

## 类型类别判断

```cpp
#include <type_traits>
#include <iostream>

static_assert(std::is_same<int, int>::value, "int == int");
static_assert(!std::is_pointer<int>::value, "int is not pointer");
static_assert(std::is_class<std::string>::value, "string is class");
static_assert(std::is_integral<int>::value, "int is integral");
static_assert(std::is_floating_point<double>::value, "double is floating point");
static_assert(std::is_array<int[10]>::value, "int[10] is array");
static_assert(std::is_enum<E>::value, "enum type check");
static_assert(std::is_function<void()>::value, "function type check");

// 编译时条件选择
template<typename T>
typename std::conditional<std::is_integral<T>::value, T, double>::type
sum(T a, T b) {
    return a + b;
}
```

## 字符串

```cpp
std::string s1 = "hello";
std::string s2 = " world";
std::string s3 = s1 + s2;  // "hello world"

s1.size();      // 5
s1.substr(0, 3);  // "hel"
s1.find("ll");    // 2

// C++17 字符串视图（不拷贝）
std::string_view sv1 = "hello world";

```

## 联合体与结构体

```cpp
// 联合体：所有成员共享同一内存
union Data {
    int i;
    double d;
    char c;
};

Data data;
data.d = 3.14;      // 设置 d
double val = data.d; // 读取 d
data.i = 42;        // 现在 i 和 d 共享内存（覆写 d 的位）

// 匿名联合体
struct Variant {
    enum Type { INT, DOUBLE, STRING };
    Type type;
    union {
        int i;
        double d;
    };
};

// 结构体默认公开成员
struct Point {
    int x;
    int y;
};
Point p{10, 20};  // 聚合初始化
```

## nullptr（C++11）

```cpp
int* p1 = nullptr;    // 推荐：类型安全的空指针
int* p2 = NULL;       // 传统：可能与整数混淆
int* p3 = 0;          // 也可，但不如 nullptr 清晰

void foo(int);
void foo(char*);

foo(nullptr);  // 调用 foo(char*)，不会与整数混淆
```

## std::variant（C++17）

类型安全的联合体替代：

```cpp
#include <variant>
#include <string>

std::variant<int, double, std::string> v = 42;
v = 3.14;
v = "hello";

if (auto p = std::get_if<int>(&v)) {
    std::cout << "int: " << *p << std::endl;
} else if (auto p = std::get_if<double>(&v)) {
    std::cout << "double: " << *p << std::endl;
}

// 访问方法二：std::visit
std::visit([](auto&& arg) {
    std::cout << arg << std::endl;
}, v);
```
