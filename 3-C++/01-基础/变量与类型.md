# 变量与类型

## 静态类型与类型推导

- C++ 是静态类型语言，编译时确定类型
- `auto` 关键字自动推导类型（C++11）

```cpp
auto x = 42;          // int
auto y = 3.14;         // double
auto name = "hello";  // const char*
auto vec = std::vector<int>{1, 2, 3};  // std::vector<int>
```

## 基本类型

| 类型 | 说明 | 大小 |
|------|------|------|
| `int` | 整数 | 通常 4 字节 |
| `long long` | 长整数 | 8 字节 |
| `double` | 双精度浮点 | 8 字节 |
| `float` | 单精度浮点 | 4 字节 |
| `char` | 字符 | 1 字节 |
| `bool` | 布尔值 | 1 字节 |

## 类型转换

```cpp
int a = 10;
double b = static_cast<double>(a);    // 编译时转换

// 基类派生类转换（运行时检查）
Base* base_ptr = new Derived();
Derived* derived_ptr = dynamic_cast<Derived*>(base_ptr);

// const 移除
const_cast<int*>(&a);

// reinterpret 底层位模式重解释
reinterpret_cast<int*>(some_ptr);
```

## 字符串

```cpp
std::string s1 = "hello";
std::string s2 = " world";
std::string s3 = s1 + s2;  // "hello world"

s1.size();      // 5
s1.substr(0, 3);  // "hel"
s1.find("ll");    // 2
```
