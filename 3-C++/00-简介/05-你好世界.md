# 你好世界

让我们从最经典的程序开始——在屏幕上输出 "Hello, World!"（你好，世界）。

## 完整代码

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

## 逐行解释

### 第 1 行：`#include <iostream>`

```cpp
#include <iostream>
```

这一行是一个**预处理指令**，告诉编译器："我需要使用输入输出功能，请帮我把 iostream 库的内容加进来"。

`#include` 就像去图书馆借书——你需要什么功能，就 `include` 什么头文件。

- `<iostream>` 是 C++ 标准库的头文件，提供了 `cin`（输入）和 `cout`（输出）功能
- 注意：没有 `.h` 后缀，这是 C++ 风格的头文件

### 第 2 行：空行

空行用于分隔代码逻辑，让程序看起来更清晰。它对程序执行没有任何影响。

### 第 3 行：`int main()`

```cpp
int main() {
```

这是**主函数**，通常每个 C++ 程序有一个 `main` 函数，通过不同翻译单元可以包含多个。程序运行时，会从 `main` 函数的第一行开始执行。

- `int`：表示这个函数会返回一个整数
- `main`：函数名字，必须这样写
- `()`：空的参数列表，表示不需要命令行参数
- `{`：函数体开始

### 第 4 行：输出语句

```cpp
    std::cout << "Hello, World!" << std::endl;
```

这行代码的作用是：**把 "Hello, World!" 显示到屏幕上**。

逐个解释：
- `std::cout`：标准输出流，通常指屏幕
  - `std::`：命名空间，表示这是标准库的内容
  - `cout`：character output，字符输出
- `<<`：流插入运算符，把右边的内容"流"向左边
- `"Hello, World!"`：要输出的文本（字符串）
- `std::endl`：换行符，让光标移到下一行
  - `endl`：end line，结束这一行

**类比**：可以把 `<<` 想象成一根管道，把数据 "流" 到屏幕上显示出来。

### 第 5 行：`return 0;`

```cpp
    return 0;
```

表示程序正常结束，返回值 0。

- `return`：返回关键字
- `0`：返回码，0 通常表示"没有错误"

### 第 6-7 行：结束括号

```cpp
}
```

闭合函数体的右花括号。

## 运行程序

将代码保存为 `hello.cpp`（`.cpp` 是 C++ 源文件的标准扩展名），然后编译运行：

```bash
g++ hello.cpp -o hello
./hello
```

你应该会看到：

```
Hello, World!
```

## 简化写法

初学者经常会看到这样的写法：

```cpp
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

`using namespace std;` 的作用是告诉编译器："我直接使用标准库的东西，不需要每次都写 `std::` 前缀"。

这种写法在简单程序中很常见，但随着程序规模增大，`std::` 前缀可以避免命名冲突，是更好的实践。

## 小结

| 代码 | 作用 |
|------|------|
| `#include <iostream>` | 引入输入输出功能 |
| `int main()` | 程序入口 |
| `std::cout << ...` | 输出到屏幕 |
| `std::endl` | 换行 |
| `return 0;` | 正常退出 |

恭喜你完成了第一个 C++ 程序！
