# C++ 语言简介

## 历史背景

C++ 由丹麦计算机科学家 **Bjarne Stroustrup** 于 1979 年在贝尔实验室工作时创造。最初这门语言叫 "C with Classes"（带类的C），在 1983 年正式更名为 C++。

Bjarne Stroustrup 设计的初衷是在C语言的基础上，添加面向对象的特性来解决大型软件开发中的复杂性问题。

## 与C语言的关系

C++ 与 C 语言有着密不可分的关系：
- **兼容继承**：几乎所有有效的 C 程序都是有效的 C++ 程序
- **共同血脉**：都提供对硬件的直接控制，语法风格相近
- **不同哲学**：C 是过程式编程，C++ 在此基础上支持面向对象、泛型编程等多范式

## 名称由来

C++ 中的 "++" 是 C 语言的自增运算符（将变量值加1）。所以 C++ 可以理解为 "C 语言的下一个阶段"，也暗示它在 C 语言基础上的增强和进化。

## C++ 的演进

| 年份 | 标准 | 重要特性 |
|------|------|----------|
| 1983 | C++ 诞生 | 类、虚函数、引用 |
| 1998 | C++98 | 第一个国际标准，STL |
| 2011 | C++11 | 现代 C++ 元年，智能指针、lambda |
| 2014 | C++14 | 泛型 lambda、变量模板 |
| 2017 | C++17 | 结构化绑定、if constexpr |
| 2020 | C++20 | 概念（Concepts）、协程、模块 |
| 2023 | C++23 | 更多提案，持续进化 |

C++ 经过数十年的发展，始终保持着对性能的最高追求，同时不断引入现代编程范式，是一门既经典又充满活力的语言。
