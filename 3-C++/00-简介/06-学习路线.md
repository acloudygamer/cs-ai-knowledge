# C++ 学习路线

## C++ 标准演进史

C++ 有着悠久的历史，每一次标准更新都带来重大变化：

| 标准 | 年份 | 里程碑意义 |
|------|------|-----------|
| C++98 | 1998 | 首个国际标准，STL 标准化 |
| C++03 | 2003 | 修正缺陷，稳定性提升 |
| **C++11** | 2011 | **现代 C++ 元年**：智能指针、lambda、范围for、正则表达式 |
| C++14 | 2014 | 泛型 lambda、变量模板 |
| C++17 | 2017 | 结构化绑定、if constexpr、文件系统 |
| C++20 | 2020 | 概念（Concepts）、协程、模块、范围库大幅增强 |
| C++23 | 2023 | std::print、更多 constexpr |

## 建议的学习顺序

### 第一阶段：语法基础（1-2周）

先掌握 C++ 的核心语法，不要被现代特性分散注意力：

1. **开发环境搭建**：安装编译器，学会编译运行
2. **变量与数据类型**：int、float、double、char、bool
3. **控制流程**：if/else、switch、for、while、do-while
4. **函数**：声明、定义、参数传递、返回值

### 第二阶段：面向对象（2-3周）

C++ 的核心价值所在：

1. **类和对象**：定义类、创建对象、访问权限
2. **构造与析构**：理解对象生命周期
3. **继承与多态**：代码复用和接口抽象
4. **运算符重载**：让自定义类型支持运算符操作

### 第三阶段：现代 C++（2-3周）

**必须学习！** 现代 C++ 与传统 C++ 有巨大差异：

1. **智能指针**：`std::unique_ptr`、`std::shared_ptr`
2. **auto 与 decltype**：类型推导，减少重复书写
3. **lambda 表达式**：匿名函数，简化回调
4. **范围 for 循环**：更安全的遍历
5. **移动语义**：`std::move`、`右值引用`

### 第四阶段：标准库与实践（3-4周）

1. **STL 容器**：`vector`、`map`、`set`、`queue`
2. **STL 算法**：排序、查找、变换
3. **string 与正则**：文本处理
4. **文件 I/O**：读写文件

### 第五阶段：深入进阶（长期）

- 模板与元编程
- 内存管理与性能优化
- 并发编程
- 设计模式

## 重点：直接学习 C++11/14/17

如果你是完全初学者，可能会在网上看到很多使用旧式 C++ 写法的教程（比如 raw pointer、new/delete、手动内存管理）。

**强烈建议直接学习现代 C++（C++11 及以上）**：
- 代码更安全（智能指针防止内存泄漏）
- 更简洁（auto 减少类型书写）
- 更现代（lambda 让回调更清晰）

旧式写法是为了兼容几十年前的代码，现在已经不再推荐。

## 推荐学习资源

### 书籍
- **《C++ Primer》**：入门经典，内容全面
- **《Effective Modern C++》**：现代 C++ 最佳实践
- **《Thinking in C++》**：深入理解语言设计

### 在线资源
- [cppreference.com](https://cppreference.com)：最权威的 C++ 参考文档
- [learncpp.com](https://learncpp.com)：适合零基础的免费教程
- [C++ Core Guidelines](https://isocpp.github.io/CppCoreGuidelines/)：现代 C++ 编程规范

### 视频课程
- Coursera、慕课网等平台上有优质的 C++ 课程

## 学习建议

1. **动手实践**：每学一个概念，马上写代码验证
2. **不要死记**：理解原理最重要，语法可以随时查文档
3. **阅读源码**：学习优秀开源项目（如 fmtlib、range-v3）
4. **保持耐心**：C++ 学习曲线陡，但每一份付出都有回报

C++ 是一门值得深入学习的语言。祝你学习顺利！
