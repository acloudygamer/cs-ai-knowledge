# C++ 开发环境

## 编译器选择

C++ 代码需要通过**编译器**转换为机器可执行文件。主要有以下几种选择：

| 编译器 | 平台 | 说明 |
|--------|------|------|
| g++ | Linux/macOS/Windows (WSL) | GNU 项目开源编译器 |
| clang | macOS/Linux | 苹果主推，错误提示友好 |
| MSVC | Windows | 微软 Visual Studio 内置 |

## 快速入门：在线编译器

如果只想快速体验 C++，可以使用在线编译器，无需安装任何软件：
- **Compiler Explorer** (godbolt.org)：查看编译后的汇编代码
- **Online GDB** (onlinegdb.com)：支持多种编译器

## Windows 开发环境

### 方案一：Visual Studio
微软官方 IDE，下载 "Visual Studio Community"（免费），安装时选择 "使用 C++ 桌面开发"。

安装完成后：
1. 启动 Visual Studio
2. 选择 "创建新项目"
3. 选择 "空项目"
4. 添加 .cpp 文件，开始编写代码

### 方案二：VS Code + MinGW
1. 安装 [VS Code](https://code.visualstudio.com/)
2. 安装 [MinGW-w64](https://www.mingw-w64.org/) 编译器
3. 配置 VS Code 的 C++ 扩展

### 方案三：WSL + g++
如果你使用 Windows 10/11，可以启用 Linux 子系统（WSL），获得原生的 Linux 开发体验：

```bash
# 安装 WSL (在 PowerShell 中)
wsl --install

# 进入 Linux 环境后安装 g++
sudo apt update
sudo apt install g++

# 编译运行
g++ hello.cpp -o hello
./hello
```

## macOS 开发环境

macOS 默认已经安装了 clang 编译器（实为 apple-clang）。

打开终端，输入：
```bash
g++ hello.cpp -o hello
./hello
```

如果没有安装 Xcode Command Line Tools，系统会提示你安装。

## Linux 开发环境

大多数 Linux 发行版已经预装 g++。如果没有：

```bash
# Ubuntu/Debian
sudo apt install build-essential

# Fedora/RHEL
sudo dnf install gcc-c++

# Arch Linux
sudo pacman -S base-devel
```

## 编译运行流程

C++ 代码从文本到可执行文件需要经过两步：

```bash
# 编译：将源代码转换为目标文件
g++ -c main.cpp -o main.o

# 链接：将目标文件合并为可执行文件
g++ main.o -o program

# 或者一步到位（最常用）
g++ main.cpp -o program
```

常用编译选项：
- `-std=c++17`：使用 C++17 标准
- `-Wall`：显示所有警告
- `-O2`：开启优化
- `-g`：包含调试信息

## 第一个项目建议

对于初学者，建议从以下组合开始：
- **Windows**：Visual Studio Community
- **macOS**：Xcode + VS Code
- **Linux**：VS Code + g++

不需要追求复杂的 IDE，先掌握语言本身才是重点。
