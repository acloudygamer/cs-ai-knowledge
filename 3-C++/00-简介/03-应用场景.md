# C++ 应用场景

## 游戏开发

游戏行业是 C++ 最大的应用领域之一。主流游戏引擎如 **Unreal Engine** 和 **Unity**（核心部分）都使用 C++ 开发。

为什么游戏偏爱 C++？
- 极致性能：每帧渲染都需要大量计算
- 内存控制：游戏需要精细管理资源
- 跨平台：一次编写，编译到各平台

## 操作系统与驱动程序

操作系统内核和硬件驱动程序几乎只能用 C/C++ 编写：
- Windows、Linux、macOS 的内核
- 显卡驱动、声卡驱动
- 打印机、扫描仪等外设驱动

因为这些软件需要直接与硬件交互，不允许任何中间层带来的性能损失。

## 嵌入式与物联网

在树莓派、Arduino、单片机等嵌入式设备上，C++ 是主力语言：

```cpp
// 典型的嵌入式代码：控制 LED 闪烁
#include <Arduino.h>

void setup() {
    pinMode(13, OUTPUT);  // 设置引脚为输出模式
}

void loop() {
    digitalWrite(13, HIGH);  // 点亮 LED
    delay(1000);             // 等待1秒
    digitalWrite(13, LOW);   // 熄灭 LED
    delay(1000);             // 等待1秒
}
```

## 高频交易

金融领域的**高频交易（HFT）**系统用 C++ 实现微秒级延迟。交易员愿意为每一微秒的优化付出高昂代价，只有 C++ 能满足这种极致性能需求。

## 数据库引擎

大多数数据库管理系统用 C++ 编写：
- MySQL、PostgreSQL
- MongoDB、Redis
- SQLite

这些系统需要高效处理海量数据，同时保证可靠性和事务性。

## 编译器与虚拟机

许多编程语言的编译器和虚拟机也用 C++ 编写：
- GCC（GNU 编译器集合）
- LLVM 项目
- V8 JavaScript 引擎
- JVM（Java 虚拟机）

这形成了一个有趣的循环：用 C++ 构建工具来创建其他语言的工具。

## 其他领域

- **图形图像处理**：Photoshop、CAD 软件
- **科学计算**：物理模拟、机器学习底层
- **虚拟现实**：VR/AR 应用
- **电信设备**：路由器、交换机软件

无论应用领域如何变化，C++ 始终是"性能优先"场景的不二之选。
