# C++

> 系统编程的性能之王

## 核心特点

- **极致性能**：编译为本地机器码，无运行时开销
- **直接硬件控制**：指针操作、内存管理，掌控每一字节
- **多范式编程**：面向对象、泛型编程、过程化，随心选择
- **强大的标准库**：STL 提供丰富的容器和算法
- **现代C++**：智能指针、lambda 表达式、移动语义，安全高效

## 适用场景

`游戏开发` `操作系统` `嵌入式` `高频交易` `数据库` `编译器`

## 学习路线

C++98 → **C++11（现代C++元年）** → C++14 → C++17 → C++20 → C++23

建议直接学习 **C++11 及以上**，跳过传统 C++ 的手动内存管理写法。
