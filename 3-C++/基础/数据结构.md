# 数据结构

## STL 容器

```cpp
#include <vector>
#include <map>
#include <set>
#include <unordered_map>
#include <string>

std::vector<int> vec = {1, 2, 3};
vec.push_back(4);
int first = vec[0];

std::map<std::string, int> dict;
dict["one"] = 1;
dict["two"] = 2;

std::set<int> unique_set = {3, 1, 4, 1, 5, 9};

std::unordered_map<std::string, int> hash_map;
hash_map["key"] = 100;  // O(1) 查找
```
