# 运算符重载

## 基本概念

运算符重载让自定义类型使用与内置类型相同的语法：

```cpp
class Vec2 {
public:
    double x, y;

    Vec2(double x = 0, double y = 0) : x(x), y(y) {}

    // 重载 + 运算符
    Vec2 operator+(const Vec2& other) const {
        return Vec2(x + other.x, y + other.y);
    }

    // 重载 - 运算符
    Vec2 operator-(const Vec2& other) const {
        return Vec2(x - other.x, y - other.y);
    }

    // 重载 * 运算符（标量乘法）
    Vec2 operator*(double scalar) const {
        return Vec2(x * scalar, y * scalar);
    }

    // 重载 == 运算符
    bool operator==(const Vec2& other) const {
        return x == other.x && y == other.y;
    }
};

int main() {
    Vec2 a(1.0, 2.0);
    Vec2 b(3.0, 4.0);
    Vec2 c = a + b;    // (4.0, 6.0)
    Vec2 d = a * 2.0;  // (2.0, 4.0)
}
```

## 友元函数重载

需要访问私有成员的运算符使用友元函数：

```cpp
class Vec2 {
    double x_, y_;
public:
    Vec2(double x = 0, double y = 0) : x_(x), y_(y) {}

    // 友元声明
    friend Vec2 operator*(double scalar, const Vec2& v);
    friend std::ostream& operator<<(std::ostream& os, const Vec2& v);
};

// 成员函数外的友元实现
Vec2 operator*(double scalar, const Vec2& v) {
    return Vec2(v.x_ * scalar, v.y_ * scalar);
}

std::ostream& operator<<(std::ostream& os, const Vec2& v) {
    os << "(" << v.x_ << ", " << v.y_ << ")";
    return os;
}
```

## 常用运算符重载

### 下标运算符 []

```cpp
class IntArray {
    std::vector<int> data_;
public:
    explicit IntArray(size_t size) : data_(size) {}

    int& operator[](size_t index) { return data_[index]; }
    const int& operator[](size_t index) const { return data_[index]; }
};

IntArray arr(10);
arr[0] = 42;
int val = arr[0];
```

### 流插入/提取运算符 << >>

```cpp
class Person {
    std::string name_;
    int age_;
public:
    Person(const std::string& name, int age) : name_(name), age_(age) {}

    friend std::ostream& operator<<(std::ostream& os, const Person& p);
    friend std::istream& operator>>(std::istream& is, Person& p);
};

std::ostream& operator<<(std::ostream& os, const Person& p) {
    os << p.name_ << " (" << p.age_ << ")";
    return os;
}

std::istream& operator>>(std::istream& is, Person& p) {
    is >> p.name_ >> p.age_;
    return is;
}

// 使用
Person p("Alice", 30);
std::cout << p << std::endl;  // Alice (30)
std::cin >> p;  // 从输入流读取
```

### 自增/自减运算符 ++ --

```cpp
class Counter {
    int value_;
public:
    Counter(int v = 0) : value_(v) {}

    // 前置 ++
    Counter& operator++() {
        ++value_;
        return *this;
    }

    // 后置 ++（返回旧值）
    Counter operator++(int) {
        Counter temp = *this;
        ++value_;
        return temp;
    }

    // 前置 --
    Counter& operator--() {
        --value_;
        return *this;
    }

    // 后置 --（返回旧值）
    Counter operator--(int) {
        Counter temp = *this;
        --value_;
        return temp;
    }

    int value() const { return value_; }
};
```

### 函数调用运算符 ()

```cpp
class Adder {
    int base_;
public:
    explicit Adder(int base) : base_(base) {}

    int operator()(int a, int b) const {
        return a + b + base_;
    }
};

Adder add_10(10);
int result = add_10(5, 3);  // 18

// 常用作谓词
std::vector<int> nums = {1, 2, 3, 4, 5};
std::for_each(nums.begin(), nums.end(), Adder(10));  // 每个元素+10
```

## 类型转换运算符

```cpp
class Fraction {
    int num_, den_;
public:
    Fraction(int num, int den) : num_(num), den_(den) {}

    // 转换为 double
    explicit operator double() const {
        return static_cast<double>(num_) / den_;
    }

    // 转换为 bool
    explicit operator bool() const {
        return num_ != 0;
    }
};

Fraction f(3, 4);
double d = static_cast<double>(f);  // 0.75
if (f) { /* 非零分数 */ }
```

## 赋值运算符

```cpp
class MyString {
    char* data_;
    size_t len_;
public:
    MyString(const char* s) {
        len_ = strlen(s);
        data_ = new char[len_ + 1];
        strcpy(data_, s);
    }

    // 拷贝赋值
    MyString& operator=(const MyString& other) {
        if (this != &other) {
            delete[] data_;
            len_ = other.len_;
            data_ = new char[len_ + 1];
            strcpy(data_, other.data_);
        }
        return *this;
    }

    // 移动赋值
    MyString& operator=(MyString&& other) noexcept {
        if (this != &other) {
            delete[] data_;
            data_ = other.data_;
            len_ = other.len_;
            other.data_ = nullptr;
            other.len_ = 0;
        }
        return *this;
    }

    ~MyString() { delete[] data_; }
};
```

## 运算符重载限制

以下运算符不能被重载：

- `::` 作用域解析
- `.` 成员访问
- `.*` 成员指针解引用
- `?:` 三元条件
- `sizeof` 类型大小
- `typeid` 类型信息
- `static_cast`/`dynamic_cast` 等转型运算符

## 最佳实践

```cpp
class MyClass {
public:
    // 1. 对于双目运算符，优先使用友元函数（非成员）
    friend MyClass operator+(const MyClass& a, const MyClass& b);

    // 2. 保持运算符的原始语义
    // + 应该表示加法，不是减法

    // 3. 提供完整的运算符集
    // 如果有 ==，也应该有 !=
    bool operator==(const MyClass& other) const;
    bool operator!=(const MyClass& other) const { return !(*this == other); }

    // 4. 标量乘法应同时支持 lhs 和 rhs
    friend MyClass operator*(const MyClass& obj, double scalar);
    friend MyClass operator*(double scalar, const MyClass& obj);
};
```
