# 装饰器与元编程

## 模板元编程

```cpp
// 编译时计算阶乘
template<int N>
struct Factorial {
    static const int value = N * Factorial<N - 1>::value;
};

template<>
struct Factorial<0> {
    static const int value = 1;
};

static_assert(Factorial<5>::value == 120, "Factorial check");

// 编译时条件类型选择
template<bool Condition, typename TrueType, typename FalseType>
struct Conditional;

template<typename TrueType, typename FalseType>
struct Conditional<true, TrueType, FalseType> {
    using type = TrueType;
};

template<typename TrueType, typename FalseType>
struct Conditional<false, TrueType, FalseType> {
    using type = FalseType;
};

using T = Conditional<std::is_pointer<int*>::value, int*, int>::type;  // int
```

### 模板参数推导

```cpp
// 模板参数推导规则
template<typename T, typename U>
struct Pair {
    T first;
    U second;
};

// 推导指引（C++17）
template<typename T, typename U>
Pair(T, U) -> Pair<T, U>;

Pair p(1, 3.14);  // 推导为 Pair<int, double>

// 类模板参数推导（C++17）
std::pair p(1, 3.14);  // 自动推导
std::vector v{1, 2, 3};  // 自动推导
std::unique_ptr<int> up = std::make_unique<int>(42);  // make_unique 推断类型
```

### Variadic Templates（可变参数模板）

```cpp
// 可变参数模板
template<typename... Args>
void print(Args... args) {
    // 展开参数包
    ((std::cout << args << " "), ...);  // C++17 折叠表达式
}

// 递归展开
template<typename T>
void print_single(T arg) {
    std::cout << arg << std::endl;
}

template<typename T, typename... Args>
void print_all(T first, Args... rest) {
    print_single(first);
    print_all(rest...);
}

// 模板参数包大小
template<typename... Args>
constexpr size_t count(Args... args) {
    return sizeof...(Args);
}

int n = count(1, 2.0, "hello");  // 3
```

## 模板特化

```cpp
template<typename T>
class Stack {
    // 通用实现
};

template<>
class Stack<bool> {
    // 特化实现，使用位压缩
};

// 偏特化
template<typename T>
class SmartPtr {
    // T* 通用版本
};

template<typename T>
class SmartPtr<T[]> {
    // T[] 数组特化版本
};

// 函数模板特化（C++11 起不推荐，推荐重载）
template<typename T>
T max_value(T a, T b) {
    return a > b ? a : b;
}

template<>
const char* max_value(const char* a, const char* b) {
    return strcmp(a, b) > 0 ? a : b;
}
```

## constexpr（C++11/14/17/20）

```cpp
// C++14 constexpr
constexpr int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

constexpr int fib5 = fibonacci(5);  // 编译时计算，fib5 = 5

// C++17 constexpr if
template<typename T>
auto process(T value) {
    if constexpr (std::is_integral<T>::value) {
        return value * 2;
    } else {
        return value + value;  // 浮点情况
    }
}

// C++20 constexpr std::string (limited)
constexpr std::string_view get_name() { return "Alice"; }

// C++20 consteval：强制编译时求值
consteval int sqr(int n) { return n * n; }
constexpr int result = sqr(5);  // 编译时计算
```

## 类型特征（Type Traits）

```cpp
#include <type_traits>

static_assert(std::is_same<int, int>::value);
```

### SFINAE 与 enable_if

SFINAE（替换失败不是错误）：模板参数推导时，某些替换会导致失败而不是编译错误：

```cpp
#include <type_traits>

// SFINAE：只有当 T 可以与 int 相加时才启用
template<typename T>
auto add(T a, int b) -> typename std::enable_if<
    std::is_integral<T>::value, T>::type {
    return a + b;
}

// 使用 enable_if 替代重载
template<typename T>
typename std::enable_if<std::is_integral<T>::value, void>::type
print(T val) {
    std::cout << "Integer: " << val << std::endl;
}

template<typename T>
typename std::enable_if<std::is_floating_point<T>::value, void>::type
print(T val) {
    std::cout << "Float: " << val << std::endl;
}
```

### C++17 constexpr if 与 SFINAE

```cpp
// 替代传统 SFINAE，更清晰
template<typename T>
auto get_value(T t) {
    if constexpr (std::is_pointer<T>::value) {
        return *t;  // T 是指针类型
    } else if constexpr (std::is_array<T>::value) {
        return t[0];  // T 是数组类型
    } else {
        return t;  // 其他类型
    }
}

// 结合类型特征
template<typename T>
void process_impl(T val, std::true_type) {
    // T 是整数类型
}

template<typename T>
void process_impl(T val, std::false_type) {
    // T 不是整数类型
}

template<typename T>
void process(T val) {
    process_impl(val, std::is_integral<T>{});
}
```

### 检测模式（Detection idiom，C++17/20）

检测类型是否具有特定成员或可调用：

```cpp
#include <type_traits>

// C++20 std::detect
template<typename T, typename = void>
struct has_begin : std::false_type {};

template<typename T>
struct has_begin<T, std::void_t<decltype(std::declval<T>().begin())>>
    : std::true_type {};

template<typename T>
void generic_print(T& container) {
    if constexpr (has_begin<T>::value) {
        for (auto& elem : container) {
            std::cout << elem << " ";
        }
    }
}

// C++20 concepts 简化写法
template<typename T>
concept Iterable = requires(T t) {
    t.begin();
    t.end();
};
```
